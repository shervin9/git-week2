Lesson 4: Basic Branching

Learn how to create and switch between branches

What is a Branch?

A branch in Git is a lightweight movable pointer to one of your commits. It's essentially a way to create a new line of development that can be worked on independently.

Creating a New Branch

To create a new branch, use the following command:

git branch feature/new-feature

Switching Between Branches

To switch to a different branch, use:

git checkout feature/new-feature

Creating and Switching in One Command

You can create and switch to a new branch in one command:

git checkout -b feature/new-feature

Viewing All Branches

To see all branches in your repository:

git branch

Best Practices

  • Use descriptive branch names
  • Create branches for new features or bug fixes
  • Keep branches up to date with the main branch
  • Delete branches after they're merged
