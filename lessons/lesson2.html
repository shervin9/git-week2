Lesson 2: Branch Management

Learn how to manage your branches.

List all of your branches

There are 2 kinds of branches:

  1. Local branch
  2. Remote branch

The local branches exist only on your own system.
The remote branches are the ones which have been pushed to Github and you can see them in branch section on your Github repository

To list both, use the bellow command:

git branch -a

List your local branches

To list your local branches use the following command:

git branch

List only the remote branches

To list the remote branches, use the following command:

git branch -r

Rename the local branch

If you are on the branch you want to rename:

git branch -m new-branch-name

If you are on a different branch:

git branch -m old-branch-name new-branch-name

Update the remote branch

After renaming locally, push the new branch to the remote and delete the old branch:

git push origin -u new-branch-name
git push origin --delete old-branch-name
